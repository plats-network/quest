# [Plats Quest](https://plats.network)

## Setup

```
composer update
```

```
php artisan key:generate
```

```
php artisan migrate --seed
```

```
php artisan starter:insert-demo-data --fresh
```

```
php artisan db:seed --class=SampleQuestData
```

```
php artisan optimize
```

```
npm install
```

For Development
```
npm run dev
```

For Production
```
npm run build
```
